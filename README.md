# ShoppyGlobe

ShoppyGlobe is a React-based e-commerce web application developed as part of the Full Stack Web Development Specialization.

## Features

* Product Listing
* Product Details Page
* Search Products
* Shopping Cart
* Add to Cart
* Remove from Cart
* Increase/Decrease Quantity
* Checkout Page
* 404 Not Found Page
* Redux Toolkit State Management
* React Router Navigation
* Lazy Loading

## Technologies Used

* React.js
* Redux Toolkit
* React Router DOM
* JavaScript
* CSS
* Vite

## Project Structure

* Components
* Pages
* Redux Store
* Custom Hooks
* Styles
